## 0. 文档定位

本文档整理 Gatebreaker Arena 一期主线的游戏界面结构与低保真布局。设计以当前一期范围为准：

- 3 名一期改装机：**「霜降」/「工事」/「蓄能」**
- 每名改装机 2 条主路线，共 6 条路线
- 每套改装方案：**1 张路线专属芯片 + 5 张通用芯片**
- 对局内按 **0s / 15s / 30s / 45s** 分阶段确定性激活
- 不包含地图变量、局内道具、量子/过载事件、隐藏随机抽取

### 0.1 术语对照

| 用户常用说法 | 当前统一术语 | UI 显示建议 |
|---|---|---|
| 英雄 | 改装机 | 主要显示“改装机”，教程中可补一句“即战斗角色”。 |
| 芯片 | 芯片 / 模块 | 一期文档仍大量使用“芯片”，UI 暂用“芯片”。 |
| 签名芯片 | 路线专属芯片 | 显示为“专属芯片”。 |
| 牌组 | 改装方案 | 显示为“改装方案”。 |
| 觉醒 | 激活 | 显示为“激活”。 |
| 共鸣值 | 适配值 | 显示为“路线适配”。 |

## 1. UI 设计目标

### 1.1 核心体验目标

界面要让玩家快速理解三件事：

1. 我选的改装机想怎么赢。
2. 我选的路线会怎样改变改装机机制。
3. 我的 6 张芯片会在什么时候激活，并如何推进 M1/M2/M3。

### 1.2 信息优先级

| 优先级 | 信息 | 展示原则 |
|---|---|---|
| P0 | 当前改装机、主路线、激活进度、场上分数、剩余时间 | 始终清晰可见。 |
| P1 | 当前已激活芯片、下一张激活芯片、M1/M2/M3 状态 | 对局中可快速扫读。 |
| P2 | 芯片详细数值、路线完整说明、克制关系 | 放在详情面板或长按提示。 |
| P3 | 收藏、养成、解锁来源、版本说明 | 局外界面展示。 |

## 7. 改装方案编辑界面

### 7.1 功能

这是一期最重要的局外界面。目标是让玩家在 30 秒内完成：

1. 选择 1 张路线专属芯片。
2. 选择 5 张通用芯片。
3. 指定或确认 0s / 15s / 30s / 45s 激活顺序。
4. 确认 M1/M2/M3 是否可达。

### 7.4 关键交互

| 操作 | 反馈 |
|---|---|
| 选择专属芯片 | 更新时间轴 0s 专属位、刷新 M1/M2/M3 预览。 |
| 拖入通用芯片 | 如果类别超限或重复，显示不可放置提示。 |
| 调整激活顺序 | 更新时间轴和里程碑到达时间。 |
| 方案不合法 | “开始对局”置灰，并说明缺少项。 |
| 长按芯片 | 显示详细数值、适配路线和反制说明。 |

### 7.5 合法性提示

| 状态 | UI 提示 |
|---|---|
| 未选专属芯片 | “请选择 1 张极寒之路专属芯片。” |
| 通用芯片不足 5 张 | “还需选择 2 张通用芯片。” |
| 匹配芯片不足 4 张 | “可开局，但本方案无法在终盘触发 M3。” |
| 类别超过 3 张 | “同类通用芯片最多 3 张。” |
| 0s 初选不含匹配芯片 | “开局通用芯片至少 1 张需要匹配主路线。” |

## 10. 对局 HUD

### 10.1 设计目标

对局 HUD 不能遮挡球路。所有信息应贴边、压缩、可扫读：

- 顶部：时间、比分、阶段。
- 左/右或角落：玩家当前改装机状态。
- 底部：弹药与主动模块。
- 侧边：芯片激活时间轴。

### 10.2 横屏简图

```text
┌────────────────────────────────────────────────────────────┐
│ P1  2        00:43        1  P2             阶段：M2       │
│ 霜降/极寒                 工事/炮台                       │
│ M1✓ M2✓ M3@45s            M1✓ M2- M3@45s                   │
├────────────────────────────────────────────────────────────┤
│                                                            │
│                                                            │
│                    [ 六边形/竞技场区域 ]                   │
│                                                            │
│                                                            │
├────────────────────────────────────────────────────────────┤
│ 弹药 ● ● ○   主动模块 [冷却 8s]      下一激活 45s：折射瞄准 │
└────────────────────────────────────────────────────────────┘
```

### 10.3 边缘激活条

```text
已激活                 待激活
[专属] [A] [B] [C] [D] | 45s [E]
 M1✓        M2✓             M3?
```

### 10.4 改装机状态提示

| 改装机 | 状态信息 | HUD 位置 |
|---|---|---|
| 霜降 | 当前威胁球的低温层数 / 冻结冷却 | 球体环形特效为主，HUD 只显示简略状态。 |
| 工事 | 屏障剩余次数 / 冷却 / 电弧脉冲冷却 | 改装机状态条 + 屏障本体显示。 |
| 蓄能 | 蓄能层数 / 爆发可用 / 光芒层数 | 底部资源槽。 |

### 10.5 对局中提示事件

| 事件 | 展示 |
|---|---|
| t=15/30/45 激活芯片 | 屏幕侧边滑入小卡片 1.5s，不暂停对局。 |
| M1/M2/M3 达成 | 顶部短提示 + 角色主题音效。 |
| 冻结 / 屏障失效 / 蓄能爆发 | 场内效果优先，HUD 只做简短文字。 |

### 12.1 功能

- 告诉玩家胜负结果。
- 解释关键构筑节点：何时 M2、何时 M3、哪张芯片贡献最大。
- 发放工程点和解锁进度。
- 提供再来一局、调整方案、查看回放入口。

### 12.2 简图

```text
┌────────────────────────────────────────────────────────────┐
│ 胜利                                                       │
├────────────────────────────────────────────────────────────┤
│ 比分：3 - 1          时长：64s          工程点 +50         │
├────────────────────────────────────────────────────────────┤
│ 构筑回顾                                                   │
│ 0s  寒霜之触 + 重发球 + 长板：M1 达成                      │
│ 15s 蓄能击：M2 达成                                        │
│ 45s 折射瞄准：M3 达成                                      │
│                                                            │
│ 关键表现：冻结触发 3 次 / 低温威胁球命中率 42%             │
├────────────────────────────────────────────────────────────┤
│ [再来一局]    [调整方案]    [查看回放]    [返回主界面]      │
└────────────────────────────────────────────────────────────┘
```

### 12.3 结算指标建议

| 改装机 | 关键指标 |
|---|---|
| 霜降 | 低温累计次数、冻结次数、冰晶转化加速次数。 |
| 工事 | 屏障阻挡次数、屏障失效次数、电弧命中次数。 |
| 蓄能 | 蓄能层数峰值、爆发次数、临时复制球得分。 |

### 13.3 收藏页原则

- 收藏页服务“查资料”和“替换方案”，不承载对局前的全部决策。
- 一期未解锁芯片可显示来源，但不要阻断默认方案开局。
- 排位若使用官方租借池，应在芯片上显示“排位可用”标签。

## 14. 新手引导流程

```text
首进游戏
  ↓
基础弹球教学：移动挡板、发球、得分
  ↓
选择推荐改装机：霜降
  ↓
系统装备默认方案：极寒之路
  ↓
展示 0/15/30/45s 激活时间轴
  ↓
训练对局
  ↓
结算解释：你在 15s 达成 M2，冻结变强
  ↓
解锁方案编辑
```

### 14.1 引导限制

- 首局不要求玩家手动配 6 张芯片。
- 首局只解释 1 名改装机、1 条路线、3 个时间点。
- 首局结束后再开放完整方案编辑。

## 17. 一期界面验收标准

- 玩家能在 30 秒内完成“选改装机 → 选路线 → 确认方案 → 开局”。
- 玩家在赛前能看懂 6 张芯片的激活时间。
- 对局中玩家能在 1 秒内看到下一张芯片何时激活。
- 玩家能理解 M1/M2/M3 是由“主专属 + 匹配通用芯片数量 + 时间门槛”触发。
- 结算能解释至少一个关键胜负原因，而不是只显示胜负和奖励。
