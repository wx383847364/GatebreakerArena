01 / OVERVIEW

两大芯片类型

芯片分签名芯片（英雄专属）和通用芯片（无归属），混合组牌。

★

签名芯片

Signature Chip — 英雄专属
  • 绑定1个英雄 + 1条路径，不可跨英雄使用
  • 品级精铸 / 共鸣 / 至高（三阶）
  • 共鸣值精铸+2 · 共鸣+4 · 至高+6
  • 专属效果品级越高，附带越强的行为质变
  • 数量10英雄 × 3路径 × 3品级 = 90张
  • 可升级精铸→共鸣→至高，消耗碎片+低品级芯片
◆

通用芯片

Universal Chip — 无英雄归属
  • 类别攻 / 守 / 速 / 乱（四类）
  • 稀有度Common / Rare / Epic / Legendary
  • 共鸣值类别匹配路径时每张+1
  • 作用提供数值增强 + 辅助匹配路径共鸣
  • 数量24张（v0.1已有设计）
  • 限制每类最多3张，Legendary最多2张

05 / DECK

牌组构建规则

牌组 = 签名芯片 + 通用芯片，合计最多8张。玩家需要在编辑阶段决定"深入一条路径"还是"广撒三条路径"。

牌组限制

牌组总上限8 张
签名芯片上限最多 5 张
通用芯片上限最多 5 张
同路径签名芯片最多 1 张（取最高品级）
至高签名芯片最多 2 张
Legendary 通用芯片最多 2 张
同名芯片不可重复
通用芯片每类上限最多 3 张

叠加规则

签名品级 × 里程碑加法叠加（独立计算）
签名品级 × 通用芯片乘法叠加
通用芯片 × 里程碑加法叠加
同路径多签名不可（只取最高品级）
不同路径签名可同时装备
签名 × 英雄代价品级效果可部分抵消
签名 × 量子事件不关联
签名 × 地图变量独立叠加

07 / AWAKENING

局内觉醒流程

对局开始时觉醒3张芯片，随着时间推移共鸣波动带来更多觉醒。签名芯片有保底机制——保证每局至少体验1条路径质变。

t = 0s · 初始觉醒
3张芯片觉醒
从牌组伪随机抽取。如果牌组有签名芯片，至少1张为签名芯片（保底）。可能触发 M1 或 M2。
t = 20s · 波动觉醒
+1张芯片
共鸣波动带来第4张觉醒。可能升级 M1→M2。纯随机抽取，无保底。
t = 40s · 波动觉醒
+1张芯片
第5张觉醒。可能升级 M2→M3。共鸣值持续累积——中盘是"生长"阶段。
t = 55s · 超载觉醒
+1张芯片
超载觉醒。如果有未觉醒的签名芯片，优先抽取。最可能触发 M3 极化——终盘"绽放"。

镜像模式规则

通用芯片镜像觉醒——双方获得相同通用芯片，保证数值公平。
签名芯片各自觉醒——各自从自己的签名芯片中抽取，产生差异化体验。
每方最多2张签名觉醒，差异可控。竞技差距来自操作和策略，不是纯数值。

觉醒展示

签名芯片展示时：英雄肖像 + 路径图标 + 品级边框（精铸银 / 共鸣金 / 至高彩虹）。
签名芯片展示时间 1.2s（比通用 0.8s 长），强调仪式感。
通用芯片展示时：类别图标 + 稀有度边框（白 / 蓝 / 紫 / 金）。
