## 0. 文档目的

本文档是芯片内容的唯一索引，用于记录每张芯片的 ID、类别、效果、适用版本、状态与变更历史。

一期可用芯片总数为 **24 张**：

| 类型 | 数量 | 构成 |
|---|---:|---|
| 通用芯片 | 12 | 攻击 4 / 防守 4 / 速度 4。 |
| 路线专属芯片 | 12 | 3 英雄 × 2 路线 × 2 个侧向变体。 |
| **合计** | **24** | 不含后置与特殊模式芯片。 |

### 0.1 一期数据边界

- 本文以下数值为一期原型的**初始可配置值**，可在 Playtest 后调整，但不得改变 Chip ID、路线归属、变体取舍和适配值规则。
- 后置、V2 候选与特殊模式芯片没有可用数值，必须保持禁用状态；它们不是“不完整的一期芯片”。
- 一期通用芯片不启用 Lv1/Lv2/Lv3 升级，全部按下表的单一标准数值生效。
- 一期路线专属芯片不启用品级与升级链，全部按 +3 适配值生效。

### 0.2 一期配装与激活规则

```text
赛前方案：1 张路线专属芯片 + 5 张通用芯片

t=0   路线专属 + 2 张初选通用芯片
t=15  第 3 张通用芯片
t=30  第 4 张通用芯片
t=45  第 5 张终盘通用芯片
```

- 初选的 2 张通用芯片由玩家赛前指定，其中至少 1 张必须匹配主路线。
- 一期不使用隐藏随机抽取；剩余三张的激活顺序由玩家指定或完整展示。
- 每类通用芯片最多装备 3 张，同名芯片不可重复。
- M1：1 张匹配通用芯片；M2：2 张；M3：4 张，且不早于 t=45s。

## 1. 通用芯片（一期可用）

通用芯片只改变基础弹球层的参数，不得生成或复制低温、冻结、屏障、蓄能、限时分裂、电弧脉冲等英雄专属机制。

### 1.1 攻击类（Strike）

| Chip ID | 名称 | 一期固定效果 | 匹配路线 | 状态 |
|---|---|---|---|---|
| `STRIKE_SERVE` | 重发球 | 发球初速 +20%。 | 攻+守、攻+速 | V1 可用 |
| `STRIKE_POWER` | 蓄能击 | 己方球由挡板反弹后获得 +15% 球速；同一球每 2s 最多触发一次。 | 攻+守、攻+速 | V1 可用；v0.3 重定义 |
| `STRIKE_ANGLE` | 折射瞄准 | 挡板边缘反弹的最大偏转角 +12°。 | 攻+守、攻+速 | V1 新增 |
| `STRIKE_OVERCHARGE` | 过载 | 己方球速上限 +15%，但己方挡板长度 -8%。 | 攻+守、攻+速 | V1 可用 |

## 2. 路线专属芯片（一期可用）

### 2.1 通用规则

| 项目 | 规则 |
|---|---|
| 绑定 | 每张专属芯片绑定 1 名英雄与 1 条路线。 |
| 配装 | 方案中只能装备当前主路线的 1 张专属芯片。 |
| 适配值 | 两种变体均提供 +3 适配值；变体不改变 M3 达成速度。 |
| 积累 / 稳定型 | 更可靠地建立、保存或维持英雄核心资源；单次兑现收益较低。 |
| 兑现 / 风格型 | 强化资源转化、主动窗口或路线特色；在效率、持续时间、容错或冷却上付出明确代价。 |
| 一期公平 | 原型与早期 PvP 使用官方租借/全解锁池，不售卖影响对局强度的变体。 |
| 第 3 方向 | 不进入一期；待一期数据验证后作为 V2 候选。 |

### 2.2 「霜降」专属芯片

| 路线 | 变体 | Chip ID | 芯片名 | 方向 |
|---|---|---|---|---|
| 极寒之路 | 积累 / 稳定 | `SIG_FROST_DEEP_FREEZE_TOUCH` | 寒霜之触 | 提高低温建立效率；冻结附加收益较低。 |
| 极寒之路 | 兑现 / 风格 | `SIG_FROST_DEEP_FREEZE_DOMAIN` | 冰封领域 | 强化冻结后的减速压制；低温建立效率较低。 |

| 冰晶之路 | 兑现 / 风格 | `SIG_FROST_ICE_CRYSTAL_PRISM` | 霜化棱镜 | 转化球速更高，但加速窗口更短。 |

| Chip ID | 主强化数值 | 明确取舍 | 上限 / M2-M3 交互 |
|---|---|---|---|

| `SIG_FROST_DEEP_FREEZE_DOMAIN` | M2 冻结后减速 `-20% / 1.5s → -30% / 2s`。 | 低温命中 `+15 → +12`。 | 仅在 M2 激活后获得强化减速；M3 仍按固定终局控制结算。 |

| `SIG_FROST_ICE_CRYSTAL_PRISM` | 转化球速 `+25% → +35%`。 | 低温衰减 `5/s → 7/s`；转化持续 `3s → 2s`。 | 仍最多 2 层、总加速不超过 +50%；适合短窗口反击。 |

| `SIG_MECH_TURRET_GUIDANCE` | 反弹导向误差 `±12° → ±5°`。 | 反弹速度 `+20% → +15%`。 | M2/M3 的反攻机制不变；提高可预测的进攻角度。 |
| `SIG_MECH_TURRET_COUNTER` | 反弹速度 `+20% → +35%`。 | 屏障间隔 `18s → 20s`；持续 `8s → 6s`。 | M3 电弧脉冲仍为全场共享 3s 冷却；不因该变体额外增发。 |
| `SIG_RADIANT_CHARGE_RESERVE` | 爆发阈值 `6 → 5` 层；护盾 `0.5s → 0.7s`。 | 爆发球速 `+45% → +35%`。 | M2/M3 分裂数量不变；更适合稳定地取得爆发窗口。 |
| `SIG_RADIANT_CHARGE_JUDGMENT` | 爆发球速 `+45% → +60%`；爆发加速持续 `3s → 4s`。 | 爆发阈值 `6 → 7` 层；护盾 `0.5s → 0.3s`。 | M2/M3 分裂数量不变；强化高风险的单次兑现。 |

| `SIG_RADIANT_GLOW_AURA` | 第 1/2/3 层球速 `+8/+6/+4% → +12/+9/+6%`。 | 每层持续 `5s → 3.5s`。 | 总球速仍受 +40% 上限；高层必须持续反弹维持。 |

## 4. 版本管理规则

### 4.1 ID 与兼容性

- 已发布、已进入玩家存档或已用于线上对局的 Chip ID 不得复用至不同效果。
- 当前原型阶段允许对未发布 ID 做一次明确记录的重定义（例如 `STRIKE_POWER`）；进入测试存档后，效果重做必须保留旧 ID 并标记“后置/废弃”，新效果使用新 ID。
- 数值调整记录在本文件的变更日志中；行为变化必须同步更新关联英雄与系统文档。
- 路线专属芯片的 ID 由 `SIG_<HERO>_<PATH>_<VARIANT>` 组成。

### 4.2 状态枚举

| 状态 | 含义 |
|---|---|
| V1 可用 | 一期可在默认池、原型和测试中使用。 |
| V1 新增 | 一期新增 ID，待原型数值验证。 |
| V2 候选 | 二期视英雄与路线验证结果决定。 |
| 后置 | 不进入当前版本，保留设计与 ID。 |
| 禁止常规 PvP | 仅可作为特殊模式或 PVE 候选。 |
| 已改造 | 旧 ID 不再按原效果使用，功能迁移至新 ID。 |
