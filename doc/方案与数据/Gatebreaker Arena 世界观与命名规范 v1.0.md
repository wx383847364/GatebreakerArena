## 0. 文档定位

本文档是 Gatebreaker Arena 所有**命名、术语、叙事语言**的权威来源。当其他文档与本文档冲突时，以本文档为准。

### 为什么需要这份文档

项目初期英雄与芯片系统使用了奇幻 RPG / MOBA 风格的命名（冰雪女王、辉光圣骑、共鸣碎片、觉醒……），与弹球/打砖块核心玩法的街机体感产生风格裂缝。玩家在操作挡板时的心智模型是"我在操作一台物理机器"，但叙事层在讲"魔法英雄释放冰霜魔法"——两者认知打架。

本文档通过**统一世界观 + 机械命名体系**消除这道裂缝。

## 1. 世界观：弹门网络

### 1.1 核心设定

Gatebreaker Arena 的对战发生在一个名为**「弹门网络」(Gatebreaker Network)** 的虚拟竞技空间中。

- **时代**：近未来，虚拟竞技体育已经职业化
- **场所**：弹门网络是一个共享的数字物理空间，拥有真实的碰撞、弹道、引力规则
- **参与者**：玩家被称为**「接入者 (Runner)」**，通过终端接入网络，操控自己的数字挡板
- **机器**：每个接入者拥有一台**「改装机 (Rig)」**——一台可定制的数字弹球台，装备不同的功能模块

### 1.2 关键概念

| 概念 | 说明 | 对应现实类比 |
|------|------|------------|
| 弹门网络 | 对战发生的虚拟空间 | 电竞赛场 / VR 竞技场 |
| 接入者 | 玩家身份 | 电竞选手 |
| 改装机 | 玩家的弹球台配置 | 赛车（赛车游戏中的改装车） |
| 模块 | 安装在改装机上的功能件 | 赛车改装件 / 电脑插件 |
| 激活 | 模块在对局中生效 | 程序加载 / 设备启动 |
| 过载 | 模块触发瞬间效果 | 超频 / 爆发模式 |

### 1.3 为什么这个世界观成立

**叙事自洽**：接入者的"技能"不是魔法，是改装机上的**硬件功能**。球带寒气是因为安装了冷冻涂层模块；时间静止是阻尼力场全功率输出；引力井是预装的引力锚点。每个技能效果都有物理/技术解释。

**操作自洽**：玩家在操作挡板时，心智模型是"我在操控改装机"，而不是"我在扮演一个魔法角色"。技能触发时是"我启动了机器上的功能"，和操作挡板完全连贯。

**街机文化锚点**：弹球台改装、街机竞技、数字竞技场——这套语言和弹球/打砖块的街机血统同宗同源，不产生外来感。

**IP 延展空间**：接入者可以有个人形象（接入前的真人形象）、改装机可以有外观定制、模块有不同厂商/系列——商业化空间充足。

## 2. 命名体系原则

### 2.1 四条铁律

| 铁律 | 含义 | 违反示例 |
|------|------|---------|
| **机械优先** | 名称必须是物理/技术概念，不是魔法/超自然概念 | ❌ 冰雪女王 ✅ 冷冻涂层 |
| **效果可推断** | 玩家看到名字能猜到大概效果 | ❌ 相位转移 ✅ 穿墙协议 |
| **街机调性** | 语言风格偏向机械、电路、物理、程序 | ❌ 圣光护盾 ✅ 能量护盾 |
| **中英一致** | 中文名和英文代号传达同一意象 | ❌ 无私守护/Heavy Frame ✅ 限速框架/Slow Frame |

### 2.2 命名层级

```
接入者 (Runner)          ← 玩家身份（不命名，玩家自定义）
  └─ 改装机 (Rig)        ← 英雄身份，有代号
       ├─ 被动模块       ← 替代"被动特性"，整局生效的硬件功能
       ├─ 主动模块       ← 替代"主动技能"，手动触发的硬件功能
       └─ 代价/限制      ← 改装机的硬件约束
```

### 2.3 改装机命名格式

每台改装机有三个层级的名称：

| 层级 | 格式 | 示例 |
|------|------|------|
| **代号** | 「二字」中文 | 「霜降」「电涌」「锚点」 |
| **英文标识** | 单词，全大写或首字母大写 | CRYO / SURGE / ANCHOR |
| **内部 ID** | `HERO_英文标识`（不变，保持代码稳定） | `HERO_FROST_QUEEN` |

> **重要**：内部 ID（如 `HERO_FROST_QUEEN`）是代码层的稳定标识符，**不随显示名变化而修改**。显示名是可独立更新的本地化字符串。这是游戏工业标准实践。

### 2.4 模块（技能）命名格式

| 类型 | 格式 | 示例 |
|------|------|------|
| 被动模块 | 四字中文 + (英文) | 冷冻涂层 (Cryo Coating) |
| 主动模块 | 四字中文 + (英文) | 冷却过载 (Coolant Overload) |
| 代价 | 四字中文 + (英文) | 低温损耗 (Cryo Drain) |

避免：
- 魔法/奇幻词汇（冰霜、神圣、暗影、灵魂、审判）
- RPG 职业词汇（女王、领主、圣骑、术士、法师）
- 抽卡手游词汇（共鸣、觉醒、命运、羁绊）

## 5. 叙事语言规范

### 5.1 技能描述模板

所有技能描述应使用以下模板，避免魔法/奇幻叙事：

```
[模块名]：[改装机名] 的 [球/挡板/场地] [物理变化]，[触发条件] 时 [效果]。
```

**正确示例**：
- 冷冻涂层：「霜降」的球表面涂覆冷冻液，碰对手挡板时凝结，累积冷冻值。
- 电磁偏转：「电涌」的球命中对手挡板时释放电磁脉冲，偏转附近球路。
- 阻尼力场：「阻尼」的挡板周围生成减速力场，对手球进入时球速降低。

**错误示例**：
- ❌ 寒气灌注：冰雪女王发出的球携带寒气，对手累积寒气值。
- ❌ 链式闪电：雷霆审判者的球命中挡板时释放链式闪电。
- ❌ 时间 dilation：时之织者的挡板周围时间变慢。

### 5.2 禁用词表

| 类别 | 禁用词 | 替代方案 |
|------|--------|---------|
| 魔法/超自然 | 寒气、灼烧、圣光、暗影、灵魂、诅咒、神圣、审判 | 冷冻、热能、能量、迷彩、信号、故障 |
| RPG 职业 | 女王、领主、圣骑、术士、法师、审判者、守护者 | （用改装机代号替代） |
| 抽卡术语 | 共鸣、觉醒、羁绊、命运、召唤 | 激活、加载、配置、部署 |
| 神话意象 | 地狱火、暴风雪、奇点（黑洞）、天罚 | 过载弹、冷却过载、引力塌缩、全弹追踪 |

> **例外**：`奇点 (Singularity)` 作为物理学概念是可接受的，但建议统一为 `引力塌缩 (Gravity Collapse)` 以避免与黑洞的奇幻联想。
