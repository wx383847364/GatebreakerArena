## 0. 文档定位

本文档定义 Gatebreaker Arena 的**英雄系统**——局外养成的第一支柱。

英雄是玩家在局外选择并带入对战的**角色身份**，每个英雄拥有：
- 一个**被动特性**（整局持续生效，定义英雄风格）
- 一个**主动技能**（玩家手动触发，有冷却）
- 一个**代价/弱点**（确保竞技平衡）

英雄是芯片系统的**基底**：英雄定义"你是谁"，芯片定义"这局你额外拿到了什么"。两者乘法叠加。

### 与现有系统的层级关系

```
英雄 (身份层)     ← 你选谁，决定被动+主动+代价
  × 芯片觉醒 (随机层)  ← 每局随机抽到的修饰器
    × 地图变量 (环境层)  ← 场地结构变化
      × 局内道具 (即时层) ← 临时buff/debuff
```

四层独立运作，乘法叠加，最终形成一局对战的有效规则。

## 1. 设计哲学

### 1.1 英雄设计的四条铁律

| 铁律 | 含义 | 违反后果 |
|---|---|---|
| **每个英雄有明确代价** | 被动越强，代价越大。没有免费午餐 | 英雄碾压，非英雄玩家无体验 |
| **特性必须可感知** | 玩家在 10 秒内能说出"这个英雄干了什么" | 认知过载，玩家不会选 |
| **反制必须存在** | 每个英雄都有破解方式，不存在无敌build | 环境固化，无策略深度 |
| **PvP 排位可选镜像** | 竞技模式中双方可选相同英雄，保证公平 | 养成差距碾压实竞技 |

### 1.2 机制分类框架

10 个英雄覆盖 6 种机制原型，确保玩法多样性：

| 机制原型 | 英雄 | 核心循环 |
|---|---|---|
| **状态累积** | 冰雪女王、烈焰领主 | 反复用球打击对手 → 累积状态条 → 触发控制效果 |
| **连锁反应** | 雷霆审判者 | 球碰挡板 → 触发连锁 → 影响其他球/玩家 |
| **场地操控** | 重力使者、机械工程师 | 改变战场结构 → 影响所有球的路径 |
| **时空干扰** | 时之织者 | 减速/冻结对手球 → 掌控节奏 |
| **反伤反击** | 荆棘守护者、辉光圣骑 | 被攻击 → 积累反击资源 → 爆发反攻 |
| **隐蔽突袭** | 暗影行者 | 隐藏球路 → 对手反应不及 → 得分 |
| **概率操控** | 量子术士 | 随机传送/分裂 → 不可预测 → 制造混乱 |

### 1.3 状态累积系统（通用框架）

冰雪女王和烈焰领主共享一套**状态累积框架**，后续新英雄可复用：

```csharp
// 通用状态累积框架
public class StatusAccumulationSystem
{
    // 每名玩家维护一组状态条
    // Dictionary<PlayerId, Dictionary<StatusType, StatusState>>
}

public struct StatusState
{
    public StatusType Type;        // Frost / Heat / Radiance / ...
    public float CurrentValue;     // 当前累积值 (0~MaxValue)
    public float MaxValue;         // 触发阈值 (默认100)
    public float DecayPerSecond;   // 自然衰减速率
    public float AccumulatePerHit; // 每次命中累积量
    public TriggerEffect Trigger;  // 达到阈值时触发的效果
    public float TriggerDuration;  // 触发效果持续时间
    public float PostTriggerReset; // 触发后重置值 (通常归零)
}

public enum StatusType
{
    None,
    Frost,     // 冰雪女王：冰冻
    Heat,      // 烈焰领主：灼烧减速
    Radiance,  // 辉光圣骑：自身光辉层数（非对敌）
}

public enum TriggerEffect
{
    FreezePaddle,      // 冰冻挡板（不可移动）
    SlowPaddle,        // 减速挡板
    SpeedBoostBall,    // 球速暴涨
    ReflectBall,       // 球反弹向敌方
}
```

**累积触发条件**：

| 触发点 | 说明 |
|---|---|
| `OnPaddleBounce` | 球碰挡板时，球归属者的英雄特性施加给挡板归属者 |
| `OnWallBounce` | 球碰墙时（部分英雄） |
| `OnGoalScored` | 进球时 |
| `OnGoalRebound` | 己方门回弹时 |

**衰减规则**：状态值持续衰减，鼓励持续进攻。如果对手拉开距离不打球，状态会慢慢消退。

## 2. 十英雄完整设计

### 2.1 冰雪女王 (Frost Queen)

| 属性 | 内容 |
|---|---|
| **代号** | `HERO_FROST_QUEEN` |
| **主题色** | 冰蓝 #4FC3F7 |
| **元素** | 冰 |
| **机制原型** | 状态累积 — Frost |
| **玩家幻想** | "你越接我的球，你越动不了。" |

#### 被动特性：寒气灌注

冰雪女王发出的所有球携带**寒气**。当对手用挡板反弹她的球时，对手累积 **+15 寒气值**。寒气值达到 **100** 时，对手挡板被**冰冻 1.5 秒**（完全无法移动），冰冻结束后寒气值归零。

| 参数 | 值 [PLACEHOLDER] | 说明 |
|---|---|---|
| 每次挡板反弹累积 | +15 | 约 7 次反弹触发冰冻 |
| 触发阈值 | 100 | |
| 冰冻时长 | 1.5s | 60s 局大约触发 2~3 次 |
| 寒气衰减 | 5/s | 2s 不被碰就掉 10 点 |

**关键设计**：只有**挡板反弹**累积寒气，**墙壁反弹不累积**。这意味着对手可以选择"放球撞墙"来避免累积——但墙弹的角度不可控，球可能直接进门。

#### 主动技能：暴风雪 (Blizzard)

| 属性 | 值 |
|---|---|
| 效果 | 己方所有场上球的寒气累积量翻倍（+30/次），持续 5s |
| 冷却 | 25s |
| 初始冷却 | 8s（开局不能立刻用） |
| 视觉 | 全场飘雪粒子 + 己方球尾迹变浓白 |

**战术意图**：在已经累积了 60~70 寒气时开启，5 秒内 2 次挡板反弹即可冰冻。是"收割"节奏的加速器。

#### 代价：沉重寒冰

| 代价 | 效果 |
|---|---|
| 球初速 -10% | 寒气是沉重的，球飞得慢 |
| 球最大速度 -8% | 上限也低 |

**反制方式**：
1. 让她的球撞墙而非挡板——墙弹不累积寒气
2. 拉开节奏，利用寒气衰减窗口
3. 选时之织者/雷霆审判者等远距离英雄，减少挡板接触

### 2.2 烈焰领主 (Pyro Lord)

| 属性 | 内容 |
|---|---|
| **代号** | `HERO_PYRO_LORD` |
| **主题色** | 烈焰橙 #FF6D00 |
| **元素** | 火 |
| **机制原型** | 状态累积 — Heat |
| **玩家幻想** | "接我的球会烫手，你敢接几次？" |

#### 被动特性：灼烧印记

烈焰领主发出的所有球携带**灼烧**。对手用挡板反弹他的球时，累积 **+12 热量值**。热量达到 **100** 时，对手挡板**着火**：移动速度 **-50%**，持续 **2.5 秒**，着火结束后热量归零。

| 参数 | 值 [PLACEHOLDER] | 说明 |
|---|---|---|
| 每次挡板反弹累积 | +12 | 约 8~9 次反弹触发 |
| 触发阈值 | 100 | |
| 灼烧时长 | 2.5s | 比冰冻长，但不是完全冻结 |
| 减速幅度 | -50% | 还能动，但很慢 |
| 热量衰减 | 4/s | 比寒气慢，持续施压感更强 |

**与冰雪女王的差异化**：
- 冰雪女王 = 完全冻结但时间短（1.5s），累积快（+15/次），衰减快（5/s）
- 烈焰领主 = 减速但时间长（2.5s），累积慢（+12/次），衰减慢（4/s）

冰雪女王是"瞬间爆发控制"，烈焰领主是"持续灼烧压迫"。

#### 主动技能：地狱火 (Inferno)

| 属性 | 值 |
|---|---|
| 效果 | 下一次发球的球为**地狱火球**：球速 +50%，碰撞半径 +50%，穿透 1 次墙体碰撞（不反弹直接穿过） |
| 冷却 | 20s |
| 初始冷却 | 6s |
| 视觉 | 球体火焰包裹 + 拖尾火焰 |

**战术意图**：地狱火球是"必进"级别的压迫工具。球速快、体积大、穿墙——对手挡板着火时发一颗，基本必进。但只有 1 颗，用完就没了。

#### 代价：燃烧消耗

| 代价 | 效果 |
|---|---|
| 最大弹药 -1 | 火焰消耗资源，弹药更少 |

**反制方式**：
1. 热量衰减比寒气慢，但灼烧只是减速不是冻结——可以顶着减速继续防守
2. 地狱火球穿墙但不穿挡板——挡板仍然是最终防线
3. 烈焰领主弹药少，打完弹药后有真空期

### 2.3 雷霆审判者 (Thunder Arbiter)

| 属性 | 内容 |
|---|---|
| **代号** | `HERO_THUNDER_ARBITER` |
| **主题色** | 闪电黄 #FFD600 |
| **元素** | 雷 |
| **机制原型** | 连锁反应 |
| **玩家幻想** | "我的球不只是打你，还会电到你旁边的球。" |

#### 被动特性：链式闪电

当雷霆审判者的球命中对手挡板时，触发**链式闪电**：从被命中的挡板向最近的 1 颗球（距离 ≤ 3 单位）释放一道闪电，该球获得一个**向最近敌方球门方向的微小偏转力**（速度不变，方向偏转 5°~8°）。

| 参数 | 值 [PLACEHOLDER] | 说明 |
|---|---|---|
| 链式范围 | 3 单位 | |
| 偏转角度 | 5°~8° | 微小但可感知 |
| 链式目标 | 最近 1 颗球（不分归属） | 可能偏转己方球 |
| 冷却 | 2s（同一挡板触发后 2s 内不再链式） | 防止刷屏 |

**设计意图**：雷霆审判者不是直接控制对手，而是**扰动全场球路**。她的球打到你 → 旁边的球被电了一下改变了方向 → 可能飞向某个球门。制造"蝴蝶效应"。

#### 主动技能：雷霆审判 (Thunder Judgment)

| 属性 | 值 |
|---|---|
| 效果 | 己方所有场上球获得**雷霆加持**：球速 +50%，且每 0.5s 方向自动微调（单次最多偏转 10°）朝向最近敌方球门，持续 4s |
| 冷却 | 30s |
| 初始冷却 | 10s |
| 视觉 | 球体被闪电弧光包裹 + 球到敌方球门的淡金色引导线 |

**战术意图**：全弹齐射的追分窗口。所有己方球同时加速 + 自动追踪敌门——4 秒内制造密集得分压力。场上球越多效果越强，是"火力全开"的爆发节点。

**与被动的协同**：被动链式闪电在对手挡板附近制造球路偏转（扰动防守），主动技能让自己的球全速冲向敌门（强化进攻）。先用被动把对手节奏打乱，再开主动一波收割。

**边界说明**：
- 方向微调是渐进的（每 0.5s 最多 10°），不是瞬间转向——对手仍有反应窗口
- 微调只改方向不改速度大小，+50% 速度增益是独立的
- 球碰墙/挡板后微调方向重新计算（以当前位置到最近敌门为准）
- 场上无己方球时释放则无效（不消耗冷却）

#### 代价：法师之躯

| 代价 | 效果 |
|---|---|
| 挡板长度 -10% | 她是施法者，不是肉搏选手 |

**反制方式**：
1. 球之间保持距离，避免链式闪电传导
2. 链式闪电可能偏转己方球——雷霆审判者自己也要小心球的位置
3. 挡板短，防守面积小，直接攻她的门

### 2.4 重力使者 (Gravity Envoy)

| 属性 | 内容 |
|---|---|
| **代号** | `HERO_GRAVITY_ENVOY` |
| **主题色** | 深空紫 #7C4DFF |
| **元素** | 重力/空间 |
| **机制原型** | 场地操控 |
| **玩家幻想** | "整个场地都在我的引力之下。" |

#### 被动特性：中心引力

场地中心始终存在一个**弱引力井**（影响半径 5 单位），所有**非己方**的球经过时受到向中心的微小加速度（0.3/s²）。重力使者的己方球不受影响。

| 参数 | 值 [PLACEHOLDER] | 说明 |
|---|---|---|
| 引力半径 | 5 单位 | 约场地半径的 1/3 |
| 加速度 | 0.3/s² | 微小但可感知 |
| 影响对象 | 所有非己方球 | 己方球免疫 |

**设计意图**：对手的球飞过中心时会被微微"拉弯"。这改变了直线球路，让对手的瞄准变得困难——他们必须计算引力偏转。重力使者自己的球不受影响，可以打直线。

#### 主动技能：奇点 (Singularity)

| 属性 | 值 |
|---|---|
| 效果 | 在指定位置生成一个**强引力井**（影响半径 6 单位），强烈吸引**所有球**（包括己方），持续 4s |
| 冷却 | 28s |
| 初始冷却 | 8s |
| 视觉 | 紫色黑洞旋涡 + 引力线 |

**战术意图**：奇点是"全场的暂停键"——所有球被吸向一个点。可以用来：
- 阻止对手快攻（把球吸走）
- 聚集球后反弹出去打群攻
- 紧急防守时把飞向己方门的球吸偏

**关键**：奇点也吸己方球，使用时机要精准。

#### 代价：引力束缚

| 代价 | 效果 |
|---|---|
| 发球初速 -15% | 球要挣脱引力发射，初速更低 |

**反制方式**：
1. 被动引力井只影响中心区域，走边路可以避开
2. 奇点吸所有球，重力使者自己也会被干扰
3. 发球初速低，她的球更容易被挡板拦截

### 2.5 时之织者 (Time Weaver)

| 属性 | 内容 |
|---|---|
| **代号** | `HERO_TIME_WEAVER` |
| **主题色** | 时钟金 #BFA46F |
| **元素** | 时间 |
| **机制原型** | 时空干扰 |
| **玩家幻想** | "靠近我的挡板，时间就变慢。" |

#### 被动特性：时间 dilation

以时之织者**挡板中心**为圆心，存在一个**减速力场**（半径 5 单位 [PLACEHOLDER]）。对手的球进入这个圆形区域时，球速 **-15%**；离开区域后恢复正常速度。减速力场**跟随挡板实时移动**。

| 参数 | 值 [PLACEHOLDER] | 说明 |
|---|---|---|
| 减速幅度 | -15% | 可感知但不致命 |
| 生效区域 | 挡板中心为圆心，半径 5 单位的圆 | 与重力使者引力井半径相同 |
| 区域跟随 | 实时跟随挡板中心移动 | 挡板移到哪，减速场就在哪 |
| 影响对象 | 所有非己方球 | 己方球不受影响 |
| 进出判定 | 每 tick 检查球与挡板中心距离 | 进入即减速，离开即恢复 |

**设计意图**：时之织者是防守型英雄。她的挡板是一个"时间力场"——对手球靠近就变慢，她有更多时间反应和防守。但减速场跟随挡板移动，这意味着：

1. **主动防守**：时之织者需要用挡板主动追踪来球方向，把球"罩"在减速场内。不是站着不动就能减速全场，而是需要操作。
2. **取舍决策**：移动挡板去减速一颗球，可能暴露另一个角度的防守空档。多球时需要判断优先罩哪颗。
3. **进攻不受限**：她的球出去打人时是正常速度——减速场只影响对手球，不影响己方。

**与半场减速的关键差异**：原设计是静态的半场减速（被动生效，不需要操作），新设计是动态的跟随减速（需要主动走位）。这提高了操作上限，同时给了对手更多反制空间。

#### 主动技能：时间静止 (Time Stop)

| 属性 | 值 |
|---|---|
| 效果 | **冻结所有对手球** 1.5 秒（球原地停止），己方球继续移动 |
| 冷却 | 35s（长冷却，大招级） |
| 初始冷却 | 12s |
| 视觉 | 全场灰度滤镜 0.5s + 对手球上冰晶时钟标记 |

**战术意图**：最强的节奏打断工具。对手球飞向你的门？冻结。你想进攻？对手球停了你打。1.5 秒在 60 秒局里是巨大的窗口。

**平衡**：35 秒冷却是所有英雄中最长的，一局最多用 1~2 次。

#### 代价：时间重量

| 代价 | 效果 |
|---|---|
| 球最大速度 -10% | 时间是沉重的，球飞不快 |

**反制方式**：
1. 减速场跟随挡板移动——攻击挡板远离的一侧可以避开减速
2. 多球同时从不同角度进攻，时之织者只能罩住一个方向
3. 时间静止只 1.5s，且冷却极长——不要被一次冻结打乱心态
4. 球速上限低，时之织者的进攻球更容易追上

### 2.6 荆棘守护者 (Thorn Guardian)

| 属性 | 内容 |
|---|---|
| **代号** | `HERO_THORN_GUARDIAN` |
| **主题色** | 荆棘绿 #2E7D32 |
| **元素** | 自然/防御 |
| **机制原型** | 反伤反击 |
| **玩家幻想** | "你打我？你会后悔的。" |

#### 被动特性：荆棘反伤

当对手在荆棘守护者的门得分时：
1. 进攻方**失去 1 发弹药**（荆棘刺伤）
2. 荆棘守护者挡板长度**永久 +8%**（本局内），最多叠加 3 次（荆棘生长）

| 参数 | 值 [PLACEHOLDER] | 说明 |
|---|---|---|
| 弹药损失 | 1 发 | |
| 挡板长度增长 | +8%/次 | 相对于基础挡板长度 |
| 最大叠加层数 | 3 层 | 最多 +24% |
| 每层移速代价 | -5% | 挡板越长越笨重，累乘在基础移速上 |
| 满层移速 | 85% | 3 层后总移速 = 100% - 3×5% |
| 触发条件 | 被进球时 | 弹药损失和生长同时触发 |
| 生长是否可逆 | 否 | 本局内永久，新一局重置 |

**设计意图**：荆棘守护者让对手"进球有代价"——越挨打，她的挡板越宽，门越难攻破。
- 你得分了 → 失去 1 发弹药（长期惩罚：后续发球能力下降）
- 但同时 → 她的挡板又长大了一截（长期后果：接下来更难得分）

两重效果形成**时间不对称的压力**：进攻方前期得分容易，但每得一分都在"培养"一个更难对付的对手。到了第三层（+24% 长度），挡板几乎封住半个门宽——但代价是 15% 移速损失，快球打远侧容易漏。

**核心决策**：
- 进攻方："我要不要趁着她还瘦的时候疯狂得分？还是控节奏别让她太快长满？"
- 荆棘守护者："我丢分了，但挡板长了——这值得。现在用加宽的挡板配合荆棘装甲反击。"

#### 主动技能：荆棘装甲 (Thorn Armor)

| 属性 | 值 |
|---|---|
| 效果 | 持续 6s 内，任何碰荆棘守护者挡板的球以 **+80% 速度**反弹，且方向**偏向最近敌方球门** |
| 冷却 | 22s |
| 初始冷却 | 6s |
| 视觉 | 挡板长出荆棘刺 + 绿色粒子 |

**战术意图**：荆棘装甲开启时是最佳反击窗口。对手球打过来 → 超高速反弹 + 朝对手门飞 → 可能直接得分。6 秒足够挡 2~3 次球。

#### 代价：迟缓根须

| 代价 | 效果 |
|---|---|
| 发球初速 -20% | 她是防御型英雄，进攻能力弱 |

**反制方式**：
1. 荆棘装甲 6 秒——等它结束再进攻
2. 弹药多时不怕被扣弹药——扣一发不影响多球压制
3. 荆棘生长让挡板越来越宽但越来越慢——前期快打远侧，后期瞄准远侧端点
4. 生长不可逆——打定主意趁她 0 层时疯狂得分，拖到 3 层后多用小球角度刁钻攻远侧

### 2.10 量子术士 (Quantum Mage)

| 属性 | 内容 |
|---|---|
| **代号** | `HERO_QUANTUM_MAGE` |

#### 被动特性：量子隧穿

量子术士的球在**墙壁反弹**时，有 **15% 概率**发生**量子隧穿**：球不反弹，而是瞬移 1~2 单位距离到反弹方向的延长线上（相当于"穿"了一小段距离后继续直线飞行）。

|---|---|---|

| 隧穿距离 | 1~2 单位 | 伪随机 |

| 视觉 | 球闪烁青色 + 残影 | |

**设计意图**：量子隧穿让球路不可预测。对手预判球会反弹 → 球实际上"穿"了一段距离继续直线飞 → 判断失误。量子术士自己也无法控制是否隧穿，但可以利用"可能隧穿"的心理威慑。

#### 主动技能：量子分裂 (Quantum Split)

| 属性 | 值 |
|---|---|
| 效果 | 己方所有场上球**立刻分裂为 2 颗**（第二颗以镜像角度飞行，球速 ×0.8） |
| 冷却 | 30s |
| 初始冷却 | 10s |
| 视觉 | 球体闪烁 + 分裂动画 + 青色波纹 |

**战术意图**：瞬间让球数翻倍。3 颗变 6 颗，场面爆炸。配合量子隧穿被动，分裂后的球墙弹时还可能隧穿——完全混乱。

**球数上限保护**：分裂受 `MaxBallsInMatch` 限制，超出的球不分裂。

#### 代价：量子不稳定性

| 代价 | 效果 |
|---|---|
| 球碰墙时 8% 概率**失去归属**（变为中性球） | 中性球被任一方挡板碰触后归属该方；进球门前未被认领则不计分 |

**设计意图**：量子术士的球不稳定——碰墙后可能变成无主球在场上飘。这制造了"争球"场景：双方都要抢在对方前面用挡板碰触中性球来认领它。量子术士虽然承担了"送球给对手"的风险，但在混乱场面中他也有机会抢回自己的球——甚至抢到对手失控的球。这是"概率操控"的代价：你获得了隧穿和分裂的概率优势，但球权也变得不稳定。

**反制方式**：
1. 量子隧穿只改变位置不改变方向——预判直线延长线上的位置
2. 分裂球速低（×0.8），更好防守
3. 中性球是双刃剑——对手可以抢走它进攻，你也可以抢回来
4. 中性球进了谁的门都不计分——趁机用挡板碰它认领，而不是干等它漂走

## 4. 英雄与芯片系统的集成

### 4.1 层级关系

```
英雄 (基底)
  定义: 被动特性 + 主动技能 + 代价
  生效: 整局持续
  
× 芯片觉醒 (修饰)
  定义: 对 EffectiveMatchRule 的数值修饰
  生效: 觉醒后持续

= 最终对局规则
```

**关键原则**：英雄定义**机制**（"球带寒气"），芯片修饰**数值**（"球速 +20%"）。芯片不改变英雄的核心机制，只调节数值。

### 4.2 芯片 × 英雄交互示例

| 芯片 | 英雄 | 交互效果 |
|---|---|---|
| 蓄能击 (+反弹球速) | 冰雪女王 | 球更快，但对手挡板反弹次数可能变少（球太快不好接），寒气累积变慢 |
| 蓄能击 (+反弹球速) | 辉光圣骑 | 爆发 +100% 速 × 芯片 +20% = 总 +120%，爆发球极难防守 |
| 长板 (+挡板长度) | 暗影行者 | 暗影行者 -15% + 芯片 +25% = 净 +10%，代价被部分抵消 |
| 快装填 (+弹药回复) | 烈焰领主 | 弹药少(-1)但回复快，真空期缩短 |
| 分裂 (墙弹概率分裂) | 量子术士 | 分裂球 + 隧穿 = 极度混乱 |
| 追踪弹 (球微调向敌门) | 重力使者 | 球被引力弯 + 芯片微调 = 双重导向 |

### 4.3 交互规则

| 规则 | 说明 |
|---|---|
| 英雄代价 × 芯片加成 | 芯片加成可以部分抵消英雄代价，但不能完全消除（代价是英雄的底线） |
| 英雄被动 × 芯片效果 | 独立生效，乘法叠加（如球速 = 基础 × 英雄修正 × 芯片修正） |
| 英雄主动 × 量子事件 | 主动技能和量子事件独立计时，可同时生效 |
| 状态累积 × 芯片 | 芯片不改变累积速率（那是英雄机制），只改变球的基础属性 |

### 4.4 镜像模式下的英雄

PvP 排位使用镜像觉醒（芯片），但**英雄仍然各自选择**：

| 模式 | 英雄 | 芯片 |
|---|---|---|
| PVE | 各选 | 各自觉醒 |
| PVP 休闲 | 各选 | 各自觉醒 |
| PVP 排位 | **各选**（允许相同） | 镜像觉醒 |

英雄选择是玩家身份的表达，即使在排位中也允许不同英雄——英雄之间的平衡通过代价机制保证。

## 7. 平衡框架

### 7.1 英雄强度评估维度

| 维度 | 评估问题 | 权重 |
|---|---|---|
| 进攻压迫 | 被动+主动能制造多少得分机会？ | 高 |
| 防守稳固 | 被动+主动能阻止多少失分？ | 高 |
| 主动性 | 玩家操作空间有多大？ | 中 |
| 容错率 | 失误代价多大？ | 中 |
| 学习曲线 | 多久能上手？ | 低 |

### 7.2 英雄间平衡矩阵

每个英雄应该对至少 2 个英雄有优势，被至少 2 个英雄克制：

| 英雄 | 优势对 | 劣势对 | 理由 |
|---|---|---|---|
| 冰雪女王 | 暗影行者、量子术士 | 时之织者、重力使者 | 冰冻克制突袭型；时之织者减速球让寒气球更慢 |
| 烈焰领主 | 荆棘守护者、辉光圣骑 | 冰雪女王、暗影行者 | 灼烧克制防守型；冰雪女王同类型但累积更快 |
| 雷霆审判者 | 重力使者、量子术士 | 荆棘守护者、暗影行者 | 连锁克制多球型；挡板短被突袭型打 |
| 重力使者 | 辉光圣骑、荆棘守护者 | 雷霆审判者、暗影行者 | 引力克慢速防守型；发球慢被突袭 |
| 时之织者 | 冰雪女王、烈焰领主 | 暗影行者、量子术士 | 减速克状态累积(球慢好挡)；突袭多角度绕开减速场 |
| 荆棘守护者 | 暗影行者、量子术士 | 烈焰领主、雷霆审判者 | 反伤克突袭；弹药少被烈焰领主反扣 |
| 机械工程师 | 量子术士、暗影行者 | 重力使者、时之织者 | 屏障挡不可预测球；场地操控叠加混乱 |
| 暗影行者 | 雷霆审判者、时之织者 | 冰雪女王、荆棘守护者 | 隐身克短挡板；冰冻/反伤克制近身 |
| 辉光圣骑 | 烈焰领主、雷霆审判者 | 量子术士、暗影行者 | 蓄力克高频进攻；不可预测打乱蓄力 |
| 量子术士 | 时之织者、辉光圣骑 | 机械工程师、冰雪女王 | 混乱克规律型；屏障稳定球路 |

### 7.3 平衡红线

| 红线 | 值 | 理由 |
|---|---|---|
| 主动技能冷却下限 | ≥ 15s | 低于 15s 变成常规武器而非技能 |
| 主动技能冷却上限 | ≤ 40s | 高于 40s 一局用不了 2 次 |
| 主动技能效果时长上限 | ≤ 6s | 超过 6s 变成持续 buff 而非技能 |
| 状态累积触发时长上限 | ≤ 3s | 控制效果不能太长 |
| 单项代价上限 | ≤ -20% | 代价不能让英雄在基础玩法中残废 |
| 球数净增上限 | ≤ +5 | 分裂/多发不能让球数失控 |
| 冰冻/完全控制时长上限 | ≤ 2s | 完全不能操作的时间不超过 2s |

## 8. 系统交互矩阵

| 系统A × 系统B | 交互规则 |
|---|---|
| 英雄被动 × 芯片效果 | 乘法叠加，芯片修数值，英雄改机制 |
| 英雄主动 × 共鸣波动 | 独立计时，主动技能不受波动影响 |
| 英雄主动 × 量子事件 | 可同时生效，效果乘法叠加 |
| 冰雪女王寒气 × 道具冻结 | 冻结效果不叠加（取较长者） |
| 烈焰领主地狱火 × 芯片穿透 | 穿透次数累加（地狱火穿1次 + 芯片穿1次 = 穿2次） |
| 量子术士分裂 × 共鸣波动 | 波动觉醒芯片不影响已分裂的球 |
| 辉光圣骑光辉 × 芯片反弹增强 | 反弹增强加成在光辉爆发时额外乘算 |
| 机械工程师屏障 × 地图墙体变量 | 屏障和地图墙体独立存在，均反弹球 |
| 暗影行者潜行 × 量子事件量子坍缩 | 坍缩瞬间球恢复可见，坍缩后恢复潜行 |
| 时之织者时间静止 × 共鸣波动 | 时间静止期间波动仍触发（系统事件不受英雄影响） |
| 英雄代价 × 地图变量 | 代价作用于英雄基础属性，地图变量改变结构，独立计算 |
| 冰雪女王寒气 × 烈焰领主热量 | 不同状态类型独立累积，互不干扰 |

## 12. 风险与缓解

### 12.1 确定性风险

| 风险 | 严重度 | 缓解 |
|---|---|---|
| 状态累积多端不同步 | 高 | 状态值每 tick 纳入 checksum |
| 量子隧穿伪随机不一致 | 高 | 使用 MatchSeed 派生的确定性 PRNG |
| 机械工程师屏障位置不一致 | 高 | 屏障生成位置由 PRNG 种子决定 |
| 冰冻/灼烧计时器多端不一致 | 中 | 计时器纳入运行时状态快照 |

### 12.2 设计风险

| 风险 | 严重度 | 缓解 |
|---|---|---|
| 状态累积英雄（冰/火）过强 | 高 | 衰减机制 + 只有挡板反弹累积 + 代价 |
| 量子术士的随机性破坏竞技体验 | 中 | 8% 失去归属作为自伤风险 |
| 暗影行者潜行让新手完全看不到球 | 中 | 拖尾仍可见 + 潜行需 4 单位飞行距离 |
| 英雄间克制关系太硬导致选英雄就赢 | 中 | 克制是"优势"不是"必胜"，操作和芯片仍重要 |
| 10 个英雄同时上线平衡工作量过大 | 高 | 分 2 批：先 3 个免费验证框架，再上 7 个 |

### 12.3 工程风险

| 风险 | 严重度 | 缓解 |
|---|---|---|
| BallRuntimeState 扩展字段影响序列化 | 中 | 新字段默认值兼容旧版本 |
| 英雄效果脚本化引入复杂度 | 中 | 用枚举+参数化实现，不用动态脚本 |
| 屏障/引力井新增碰撞体影响 swept collision 性能 | 中 | 限制同时存在的额外碰撞体数量 |
| 英雄代价修改 EffectiveMatchRule 链路复杂 | 中 | 代价在 BuildEffectiveRule 时一次性应用，不中途变 |
