## 稳定决策速览

- `App.AOT` 只放宿主、资源、持久化、HybridCLR 和平台基础设施
- `App.Shared` 只放稳定 DTO / 接口 / 事件
- `App.HotUpdate` 承载正式玩法
- 静态地图或场地模板只作为模板输入，不保存运行时状态
- 正式运行时资源统一走 YooAssets
- UI 只做表现和交互，不承载奖励公式和地图生成逻辑
