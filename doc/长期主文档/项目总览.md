# 项目总览

这份文档是 Gatebreaker Arena 长期主文档体系的第一入口。

## 项目定位

- 项目名称：Gatebreaker Arena
- 中文暂名：弹门对决
- 核心玩法：多人守护区域弹球攻防，玩家用挡板防守己方区域并把己方弹球送入对手守护区域得分
- 正式业务分层：`App.AOT / App.Shared / App.HotUpdate`

## 当前阶段

- 当前阶段：进行中
- 当前主线：待补充
- 当前建议起点：Agent 4：UI 与验证

## 当前稳定结论

- `App.AOT` 只负责宿主、资源、持久化、HybridCLR、平台桥接和基础设施。
- `App.Shared` 只放跨层稳定 DTO、接口和事件。
- `App.HotUpdate.GatebreakerArena` 是后续玩法唯一默认业务根。
- UI 只做表现和输入转发，不承载得分、发球、AI、同步或结算规则。
- 正式运行时资源统一走 YooAssets，不新增 `Resources.Load`。
